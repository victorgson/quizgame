# A simple website to play quiz game using [C# Backend](https://github.com/victorgson/team1-backend_csharp-quiz_api)

1. Clone https://github.com/victorgson/team1-backend_csharp-quiz_api and follow instructions.
2. Clone this project, run npm install
3. Might have to change port number in App.js if the backend runs on different port.
4. Run command `npm start``
5. Enjoy
